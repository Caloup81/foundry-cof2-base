Les Jets Principaux

A de nombreux moments, le système ouvrira des fenêtres pour effectuer des jets de Dés qui vous permettront de modifier les valeurs du Jet pour l’adapter à chaque situation. Ces fenêtres sont aussi le moyen de vérifier que vous avez bien activé tous les éléments de votre Personnage qui doivent vous servir dans la situation (Capacités, Équipements…).

Chaque fois que vous faites une action, avec ou sans fenêtre intermédiaire, un Message sera affiché dans le tchat de FoundryVTT. Ces messages vous donnent beaucoup d’informations sur le Jet et aussi des options pour poursuivre les Actions (appliquer les Dommages, utiliser un Point de Chance, activer ou désactiver la Capacité…).

Dans ce chapitre nous allons voir les éléments présents sur les fenêtres ainsi que sur les Messages du Chat.

Options

En fonction des @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.Xr7GWCaNfcxZlsnY#options-du-systeme]{Options du Système} sélectionnées, les fenêtres seront parfois différentes :

@UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.Xr7GWCaNfcxZlsnY#afficher-la-difficulte]{Afficher la difficulté}

En fonction de l’option choisie, le paramètre de Difficulté s’affichera seulement pour le MJ ou pour tout le monde. Ainsi, la fenêtre pourra être différente en fonction de votre statut dans la partie et les informations du Message du Chat ne seront pas les mêmes également.

@UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.Xr7GWCaNfcxZlsnY#activer-les-jets-combines-attaque-et-dommages]{Activer les jets combinés Attaque et Dommages}

De la même manière, cette option combinera une fenêtre d’Attaque et une fenêtre de Dommages si elle est activée. Dans ce cas, les Dommages pourront être lancés automatiquement si le Jet d’Attaque est réussi et ainsi afficher deux Messages dans le tchat directement.

Les options tactiques pour les attaques sont disponibles uniquement si cette option est activée.

@UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.Xr7GWCaNfcxZlsnY#afficher-les-boutons-de-dommages]{Afficher les boutons de Dommages} pour tout le monde

Si vous n’activez pas cette option, les joueurs ne verrons pas et ne pourront pas interagir avec les boutons de Dommages dans le Messages du Chat. Dans tous les cas le MJ verra les boutons et pourra les utiliser pour appliquer les Dommages sur les cibles.


Pour expliquer les fenêtres et les Messages du Chat, nous détaillerons chaque section comme si vous aviez activé la difficulté pour tout le monde et les jets combinés attaque et dommages car cela ajoute des paramètres.

Jet de caractéristique

Fenêtre du Jet

En cliquant sur le modificateur de caractéristique dans @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.9Bn7F4sxyKm3TX74#entete]{Entête} de la @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.9Bn7F4sxyKm3TX74#feuille-personnage]{Feuille Personnage} ou la @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.9Bn7F4sxyKm3TX74#feuille-rencontre]{Feuille Rencontre} , la fenêtre pour effectuer un jet de cette Caractéristique apparait.

Il est aussi possible de réaliser un test de caractéristique depuis un journal ou un description grâce à un lien personnalisé. Les @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.hAcXlQLmfEHtqwSS#demande-de-jet-dans-un-journal-(@test)]{Demande de jet dans un journal (@Test)} permet de préparer dans le texte de la campagne les jets de compétences demandés aux personnages et les conséquences de la réussite ou de l'échec du test (application de statuts, Dommages ou une description). Dans la description d'un Objet, cette fonctionnalité permet aussi de gagner du temps et de la fluidité durant la partie.

Cette fenêtre de couleur bleu pour la distinguer dans Attaques et des Dommages reprend toutes les informations (bonus, malus, difficulté…) pour la caractéristique choisie.

Type de jet

Les trois cercles permettent de voir et de modifier le type de jet. Si des éléments de votre personnage vous octroient un dé bonus ou un dé malus, le type de jet sera automatiquement ajusté mais il est toujours possible de le modifier sur cette fenêtre.

Difficulté

Si vous connaissez la difficulté pour le test de caractéristique que vous faites, inscrivez-le dans la case Difficulté. Contrairement aux jets d’attaque, ce champ n’accepte que des nombre entiers positifs. Vous ne pouvez pas écrire de variables (@oppose.atc, @cible.def…) car cela interviendra pour les jets en opposition que nous aborderons un peu plus loin.

Jet opposé

Cette case à cocher permet de réaliser un test opposé contre un token ciblé avant de réaliser le jet de compétence. Choisissez la compétence que la cible doit réalisé dans la liste déroulante qui apprait à la place de la case de difficulté.

Jet visible par

En sélectionnant une option de visibilité des jets au dessus du cadre d’écriture du Chat, ce champ s’adaptera automatiquement à ce choix. Vous avez la possibilité de modifier la visibilité du jet malgré ce choix grâce à cette liste déroulante.

Formule

Sur la ligne vous voyez la formule réalisée pour le Jet de Caractéristique. La formule est modifiable avec une bonne partie @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.a0cY3q3XV4o8Y13V#les-variables]{des variables} de votre Personnage et des formules de jets de dés. Vous pouvez ajouter, par exemple, +1d4°, +@agi, +@niv…

De plus, la formule est modifiée automatiquement par le type de jet choisi. Si vous modifiez sur la fenêtre, le type de jet, cela annulera les modifications réalisées sur la formule. Modifiez donc le type de jet en premier.

Les bonus/malus

Les deux champs "bonus" et "malus" vous permettent de rajouter à la main des bonus/malus circonstanciels. Utilisez la molette de la souris pour modifier ces valeurs. Les nombres doivent être positifs pour la case Bonus et négatifs pour la case Malus. Si vous écrivez "10" dans la case malus, cela applique "+10" au test.

Bonus total

Ce champ additionne tous les bonus/malus ajoutés grâce aux compétences sélectionnées dans la troisième section. Voir plus bas pour plus d’informations.

Critique

Dans de rares situations, un Jet de Caractéristique pourra avoir une plage de critique plus grande que le seul 20 naturel. Notez simplement dans ce champ le chiffre minimal qu’il faut faire sur le D20 pour obtenir une réussite critique (aucune restriction ne limite ce chiffre).

Modificateurs des actions

Comme nous l’avons vu dans le paramétrage des @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.a0cY3q3XV4o8Y13V]{Actions et Effets}, les Modificateurs de type @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.a0cY3q3XV4o8Y13V#jet-de-competence]{Jet de compétence} permettent de rajouter des bonus aux jets de Caractéristiques en fonction des situations.

C’est ici que les Modificateurs actifs de votre personnage apparaîtront pour chaque Caractéristique sous forme de liste. Vous voyez, le bonus accordé, le nom de la Capacité et de la Voie ainsi que les Notes écrites pour le Modificateur. Il vous suffit de cliquer sur les bonus de Compétences pour les activer pour ce jet. Les Bonus actifs deviennent rouges.

La somme des bonus activés est notée dans le champ Bonus total comme abordé précédemment.

Message du Chat

Le Message du Chat pour un Jet de Caractéristique est très simple. Il affiche le nom de la caractéristique et le résultat du Jet. Si vous affichez la Difficulté, il affichera également la Difficulté et si le test est un échec critique, un échec, une réussite ou une réussite critique.

Difficulté et utilisation de Point de Chance

Si vous afficher la difficulté et que votre Jet est un Échec, une image de trèfle apparaîtra à droite de votre résultat dans le Message du Chat si vous disposez d’au moins un Point de Chance.

En cliquant sur cette icône, vous utilisez un PC pour augmenter de 10 points votre résultat. Si vous obtenez de ce fait un succès, le Message du tchat sera modifié.

Résultat des Dés

En bas de chaque Message du Chat, une ligne affiche la formule lancée. En cliquant dessus vous pouvez déployer le résultat de chaque dé lancé.

Normalement le dé est gris mais il devient vert s’il a fait le résultat maximal ou rouge pour le résultat minimal.

Vérifier le résultat de chaque dé permet de recalculer le résultat mais surtout de rajouter des bonus si vous possédez une Capacité qui dit, par exemple, "chaque fois qu’un dé obtient le résultat minimal, considérez-le comme le chiffre maximal du dé".

Compétence(s) utilisée(s)

Si vous avez sélectionné des bonus de compétence au moment de faire le jet, ils seront listés en bas du message dans le tchat.

Jet d'attaque

Fenêtre du Jet

Certains éléments de la fenêtre d’Attaque et de Dommages sont identiques avec le Jet de Caractéristique. La couleur rouge de la fenêtre vous montre que vous réalisez une Attaque et/ou des Dommages contrairement à la fenêtre de Caractéristique.

Les paramètres de l’Attaque et des Dommages de votre action sont déjà renseignés dans la fenêtre mais il est possible de modifier chaque élément pour s’adapter à la situation lors de vos parties.

Type de jet

Les trois cercles permettent de voir et de modifier le type de jet. Si des éléments de votre personnage vous octroient un dé bonus ou un dé malus, le type de jet sera automatiquement ajusté mais il est toujours possible de le modifier sur cette fenêtre.

Le type de Jet est modifié par l’activation de la case à cocher "Temporaire" de la ligne du Jet de Dommages.

Difficulté

Il y a plusieurs possibilités qui apparaissent dans ce champ en fonction de la configuration de l’action. Si rien ne s’affiche, c’est soit que vous n’avez ciblé personne soit que la difficulté n’est pas définie dans l’Action.

  • Si vous voyez un nombre, cela indique que la difficulté est "fixe" comme par exemple @cible.def qui reprend le score de Défense de la cible.
  • Si le champ affiche @oppose.X, cela signifie que vous faites un jet en opposition. Ce type de jet sera expliqué plus bas.

Une info-bulle vous indique la formule inscrite dans la configuration de l’action.

Jet visible par

Ce champ reprend la formule inscrite dans la configuration de l'Action modifiée par le type de Jet, expliqué plus haut. Vous pouvez modifier la formule de l'Attaque mais vos modifications seront annulées si vous changez le type de Jet ensuite (et donc aussi le choix des Dommages Temporaires). Commencez toujours par ce paramètre avant de modifier la formule à la main. Vous pouvez ajouter des nombres, des variables et des formules de Jet de Dés à cette formule. Une info-bulle, vous informe de la formule inscrite dans la configuration de l'Action et des Bonus éventuels.

Jet d’Attaque

Ce champ reprend la formule inscrite dans la configuration de l’action modifiée par le type de Jet, expliqué plus haut. Vous pouvez modifier la formule de l’Attaque mais vos modifications seront annulées si vous changez le type de Jet ensuite (et donc aussi le choix des Dommages Temporaires). Commencez toujours par ce paramètre avant de modifier la formule à la main. Vous pouvez ajouter des nombres, des variables et des formules de Jet de Dés à cette formule.

Une info-bulle vous informe de la formule inscrite dans la configuration de l’action et des Bonus éventuels.

Bonus et Malus

Les deux champs "bonus" et "malus" vous permettent de rajouter à la main des bonus/malus circonstanciels. Utilisez la molette de la souris pour modifier ces valeurs. Les nombres doivent être positifs pour la case Bonus et négatifs pour la case Malus. Si vous écrivez "10" dans la case malus, cela applique "+10" au test.

Attention : les cases sont modifiées par le choix de l’Option Tactique, que nous verrons un peu plus bas. Si vous modifiez ces cases puis que vous changez d’option tactique, vos modifications seront annulées. Commencez toujours par choisir le Type de Jet et l’Option tactique avant de modifier à la main la formule ou les cases Bonus/Malus.

Critique

Un Jet d’Attaque pourra avoir une plage de critique plus grande que le seul 20 naturel. Le chiffre noté correspond au nombre minimal que vous devez faire sur le D20 pour obtenir une Réussite Critique.

Ce nombre est déjà modifié par les Actions/Effets que vous avez activées sur votre personnage. Notez simplement dans ce champ le chiffre minimal qu’il faut faire sur le D20 pour obtenir une réussite critique (aucune restriction ne limite ce chiffre).

Une info-bulle informe des effets qui influence la plage de critique.

Modificateurs de compétences

Si le test de la résolution d'Attaque est une compétence (@for par exemple), les modificateurs des Actions/Effets de l'Acteur seront listés et sélectionnable ici comme pour un test de Compétence.

Jet de Dommages

Si vous avez activé les jets combinés Attaque et Dommages dans les @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.Xr7GWCaNfcxZlsnY#options-du-systeme]{Options du Système}, à cet endroit, la ligne du test de Dommages s’affiche. Toutes les explications sur cette ligne sont détaillées plus bas.

Options Tactiques

Comme expliqué dans le Livre de Règles @UUID[Compendium.cof2-base.cof-2-base-regles.JournalEntry.SFIZ9i9XEt3gyYUC.JournalEntryPage.Dw9Ze41hgZ5MdKg2#options-tactiques]{Options tactiques}, les joueurs peuvent, ou non, utiliser une option tactique pour augmenter leur chance de toucher ou leurs Dommages. De base, aucune Option Tactique n’est sélectionnée. En cliquant sur les cercles vous pouvez appliquer les Bonus et Malus prévus dans les Règles pour chaque option. Cela modifie la formule du Jet de Dommages et l’une des cases Bonus/Malus. Pour chaque option, une info-bulle vous informe sur l’effet de cette option tactique.

Les Cibles de l’Attaque

Ici apparait le ou les Tokens que vous ciblez au moment de l’Attaque. Une info-bulle vous informe sur le nom du Token.

Si la difficulté de l'attaque est @cible.def alors la DEF de la ou des cibles sera inscrite en dessous du portrait de la cible.

Lorsque vous sélectionnez plusieurs cibles, chaque cible est évaluée individuellement. La Défense de chaque cible est comparée au résultat du Jet d’Attaque pour déterminer si l’attaque touche cette cible ou non. Ainsi, une même attaque peut toucher certaines cibles et en rater d’autres.

Le Message du Chat affichera une liste de cibles avec, pour chacune, le résultat individuel (Succès, Succès critique, Échec ou Échec critique) ainsi que la Défense de la cible si la difficulté est affichée.

Jet en opposition

Le cas des Jets en Opposition est légèrement différent. Si votre action a une difficulté sous forme de @oppose.X (par exemple, @oppose.cha pour un test opposé à un jet de Charisme), le résultat de l’action ne sera immédiat.

Un bouton "jet opposé" apparaîtra à la place de la Difficulté dans le Message du Chat. En cliquant dessus, un jet sera effectué et le résultat de l’action apparaîtra.

En cas d’attaque multi-cibles avec jet en opposition, un bouton "Jet opposé" apparaît pour chaque cible dans le Message du Chat. Chaque défenseur peut effectuer son propre jet en opposition indépendamment.

Le défenseur peut utiliser un Point de Chance pour augmenter son jet de 10 points si le résultat de l’opposition est un succès pour l’attaquant et que le défenseur dispose de Points de Chance.

Le Message d'Attaque

Le résultat de l'Attaque est affiché dans le tchat de FoundryVTT. Ce message indique le nom de l'Équipement ou de la Capacité ainsi que le nom de l'Action utilisée. Si l'action possède un Message tchat défini (voir la section @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.a0cY3q3XV4o8Y13V]{Actions et Effets}), alors c'est ce texte qui sera affiché.

Le cadre du nom

Le cadre avec le nom de l'Objet et de l'Action/Effet est dépliable. Si vous avez un Type de Jet différent de standard et que vous utilisez une des trois Options Tactiques alors cela sera indiqué par des icônes à droite du cadre ou en toutes lettres quand le cadre est déplié.

La ligne avec la formule du test ainsi que le résultat de chaque dé est aussi présente lorsque le cadre est déplié.

Vérifier le résultat de chaque dé permet de recalculer le résultat mais surtout de rajouter des bonus si vous possédez une Capacité qui dit, par exemple, "chaque fois qu'un dé obtient le résultat minimal, considérez-le comme le chiffre maximal du dé".

Résultat de l'attaquant

Le nombre qui est affiché ensuite est le résultat total de l'attaque. Si une icône s'affiche après le résultat, cela signifie que l'attaquant peut utiliser un point de Chance pour ajouter +10 à son attaque.

La liste des cibles

Ensuite, dans une section qui est dépliable, on trouve la liste de toutes les cibles de l'attaque.

Pour chaque cible est affiché une icône représentant si l'attaque est réussie, ratée, une réussite critique ou un échec critique, l'image du token et le nom du token. A droite du nom, la difficulté aura plusieurs affichages différents :

  • Affichage de la Difficulté : si vous avez activé l'affichage de la Difficulté dans les @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.Xr7GWCaNfcxZlsnY#options-du-systeme]{Options du Système} lors votre jet d'Attaque sera comparé à la Difficulté (si elle est renseignée). Dans ce cas, le Message du tchat vous informera si votre jet est un Succès critique, un Succès, un Échec ou un Échec critique.
  • Jet en Opposition : Si la Difficulté est sous forme de "@oppose.X" alors un bouton s'affiche dans le Message du tchat. Il est nommé "Jet opposé". En cliquant sur ce bouton, le jet opposé sera effectué comme expliqué plus haut dans la section @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.7BoZ8Un0GrShNPtF#jet-en-opposition]{Jet en opposition}.
  • L'affichage d'une icône signifie que la cible peut utiliser un Point de Chance pour améliorer son résultat au test opposé. Les résultats contre chaque cible sont recalculés individuellement : certaines cibles qui étaient ratées peuvent ainsi devenir touchées. Si cela génère au moins un succès, le jet de Dommages sera lancé ou mis à jour.

Effets supplémentaires

Afin de ne pas faire des Messages dans le tchat qui prennent toute la hauteur de la fenêtre, les @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.a0cY3q3XV4o8Y13V#effet-supplementaire]{Effet Supplémentaire} ne sont pas affichés dans le Message du tchat. Lorsque l'Attaque est réalisée, il faudra regarder dans l'@UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.9Bn7F4sxyKm3TX74#onglet-effets]{Onglet Effets} pour vérifier que l'Effet Supplémentaire a bien été appliqué. L'info-bulle vous dira les Effets appliqués.

Jet de dommages

Fenêtre de Dommages

Si vous effectuez un Jet de Dommages en cliquant sur l’icône ou si vous avez @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.Xr7GWCaNfcxZlsnY#activer-les-jets-combines-attaque-et-dommages]{Activé les jets combinés Attaque et Dommages} dans les @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.Xr7GWCaNfcxZlsnY#options-du-systeme]{Options du Système}, la ligne pour les Dommages est identique.

Formule de Dommages

Dans cette case est inscrite la formule configurée dans l’action et les bonus ou malus éventuels. Il est possible de modifier à la main cette formule pour adapter le Jet à la situation en jeu.

Ce champ peut recevoir des nombres, des variables ou des formules de Jet de Dés et toutes combinaisons de ces éléments.

Attention : cette case est modifiée automatiquement par les Options Tactiques, comme expliqué plus haut. Ainsi si vous modifiez à la main cette case puis que vous sélectionnez une option tactique, vos modifications seront effacées.

Bonus/Malus

Les deux champs "bonus" et "malus" vous permettent de rajouter à la main des bonus/malus circonstanciels.

Il est possible d’écrire des nombres entiers dans les deux champs. Le champ "malus" est de base négatif que vous écriviez "10" ou "-10".

Utilisez la molette de la souris pour modifier ces champs rapidement.

Temporaire

Cette case à cocher permet de déterminer si les Dommages effectués sont des Dommages létaux ou Temporaires. Si la case est déjà cochée, c’est que votre arme inflige des DM Temporaires.

Dans tous les cas, il est possible de cocher ou de décocher cette case.

Si vous souhaitez infliger des Dommages Temporaires avec une arme qui ne le permet pas, le type de Jet sera changé en "avec dé Malus".

En infligeant les Dommages à la cible grâce aux icônes du Message du Chat, les Dommages Temporaires seront ajoutés aux "DM Temp." de la cible et le Modificateur de Force de la cible sera soustrait du total de Dommages subis.

Options Tactiques

Si vous avez activé pour l'attaque une des options tactique, vous pouvez le faire également sur la fenêtre de Dommages afin d'appliquer les bonus ou malus aux Dommages sur le test.

Les Cibles de l'Attaque

Ici apparait le ou les Tokens que vous ciblez au moment du jet de Dommages. Une info-bulle vous informe sur le nom du Token

Message des Dommages

Le Message de Dommages dans le tchat présente l’image et le nom de l’Équipement ou de la Capacité utilisé dans un en-tête dépliable. En cliquant sur l’en-tête, les détails du jet (formule et résultat des dés) sont affichés.

Total de Dommages

Le nombre affiché indique le résultat brut du Jet de Dommages, sans multiplicateur appliqué.

Temporaire

La case à cocher Temp. au niveau de la carte permet de définir si les Dommages infligés sont temporaires pour toutes les cibles. Si l’équipement est configuré pour infliger des DM temporaires, la case est automatiquement cochée. Les Dommages Temporaires sont ajoutés aux "DM Temp." de chaque cible et le Modificateur de Force de la cible est soustrait du total de Dommages subis.

Section d’Application des Dommages

La section d’application est dépliable et propose deux modes :

  • Ciblées : affiche la liste des cibles de l’attaque avec leur portrait, leur nom et les contrôles individuels. Ce mode est sélectionné par défaut si des cibles étaient ciblées lors de l’attaque.
  • Sélectionnées : permet de sélectionner manuellement des Tokens sur la scène pour leur appliquer les Dommages.

Contrôles par cible

Pour chaque cible, une ligne affiche les contrôles suivants :

  • RD : case à cocher individuelle. Cochée par défaut, elle applique la Résistance aux Dommages propre à cette cible.
  • (Soin) : les Dommages deviennent des soins pour cette cible.
  • 0 : aucun Dommage n’est infligé. Sélectionné automatiquement si l’attaque a raté cette cible.
  • x½ : la moitié des Dommages est infligée. Utile pour les cibles résistantes ou ayant réussi un jet de sauvegarde.
  • x1 : les Dommages normaux sont infligés. Sélectionné par défaut si l’attaque touche cette cible (réussite normale).
  • x2 : le double des Dommages est infligé. Sélectionné automatiquement en cas de réussite critique contre cette cible.

Le montant calculé est affiché à côté de chaque cible et tient compte du multiplicateur choisi et de la RD individuelle si elle est cochée. Il est possible de modifier le multiplicateur de chaque cible avant d’appliquer les Dommages.

Un unique bouton "Appliquer" en bas de la section applique les Dommages calculés pour toutes les cibles en une seule action. Les Effets Supplémentaires sont également appliqués à ce moment-là, individuellement sur chaque cible touchée.

Multiplicateurs de Critique

Si votre Attaque doit infliger des Dommages critiques supérieurs au double des DM, il faudra faire une manipulation supplémentaire lors de l'application des Dommages.

    Exemple pour des Dommages critique x3 : Après application des dommages avec le bouton x2 sélectionné, cliquez sur les boutons x1 pour les cibles touchées, désélectionnez la case RD et appliquez à nouveau.

Ainsi, lorsque vous devez utiliser plusieurs fois une icône de Dommages ou plusieurs icones pour une même cible, il faut décocher la case RD avant de cliquer sur la deuxième icône de Dommages.

Jet de sauvegarde

Pour les tests de sauvegarde, il n'y a pas de fenêtre qui s'affiche pour le personnage qui réalise l'action. En sélectionnant la ou les cibles et en cliquant sur l'action, un message sera directement envoyé dans le tchat.

Le message du tchat affiche la caractéristique demandée et la difficulté du test. Si une formule de Dommages est configurée sur la Résolution de type @UUID[.a0cY3q3XV4o8Y13V#sauvegarde]{Sauvegarde}, le total des Dommages est affiché sous la difficulté.

Cibles et jets individuels

Chaque cible apparaît dans une liste avec un bouton "Sauvegarde" individuel (une info-bulle indique la compétence demandée).

Le propriétaire de chaque cible (ou le MJ) peut cliquer sur le bouton pour effectuer le jet de sauvegarde de cette cible. Une fenêtre de @UUID[.#jet-de-caracteristique]{Jet de caractéristique} s'ouvre pour sélectionner les éventuels bonus à appliquer.

Le résultat du jet est affiché sur la ligne de la cible avec une icône colorée indiquant le succès ou l'échec. Si la cible échoue et dispose de Points de Chance, une icône lui permet d'utiliser un PC pour tenter de transformer l'échec en succès.

Dommages liés au test

Si une formule de Dommages est configurée sur la Résolution de type Sauvegarde, des contrôles de Dommages par cible apparaissent lorsque le jet est effectué. Ces contrôles fonctionnent de la même manière que pour le @UUID[.#message-des-dommages$1]{Message des Dommages} : RD individuelle, multiplicateurs (soin, 0, x½, x1, x2) et bouton "Appliquer".

Si l'option "Dommages moitié sur réussite" est activée dans la Résolution, une cible qui réussit son jet de sauvegarde aura le multiplicateur x½ activé pour recevoir seulement la moitié des Dommages et une cible qui échoue reçoit les Dommages complets (x1). En cas de réussite critique, la cible ne reçoit aucun Dommage (multiplicateur 0).

Effets Supplémentaires

Les @UUID[.a0cY3q3XV4o8Y13V#effet-supplementaire]{Effets Supplémentaires} de la Résolution sont appliqués individuellement sur chaque cible en fonction de son résultat de sauvegarde au moment de cliquer sur le bouton "Appliquer".

Jet de soins et Consommable

Récupération Rapide et Complète

Lorsque vous effectuez une Récupération Rapide et lorsque vous souhaitez dépenser votre DR après une Récupération Complète , un Message de Chat apparait.

Il vous informe du nombre de Points de Vigueur que vous récupérez. Ces PV sont ajoutés automatiquement à votre personnage. Il n’y a donc aucune icône pour appliquer les soins dans le Message du Chat.

Un deuxième message uniquement pour le MJ résumera la récupération effectuée.

Action de Soin

Si vous consommez une potion ou tout autre Capacité ou Équipement qui possède une Action avec une @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.a0cY3q3XV4o8Y13V#resolutions]{Résolutions} de type @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.a0cY3q3XV4o8Y13V#soin]{Soin}, un Message Chat sera affiché.

Il vous indique le résultat du Jet de Soin et les Points de Vigueurs sont ajoutés automatiquement à votre Personnage ou à la cible sélectionnée si vous avez configuré la Résolution avec une cible différente de "Soi-même".

Un deuxième message uniquement pour le MJ résumera le soin effectué.

Action Consommable

Si vous définissez une Action avec une Résolution de type @UUID[Compendium.cof2-base.cof-2-guide-du-systeme.JournalEntry.Cj2L7dqSXItRSNXi.JournalEntryPage.a0cY3q3XV4o8Y13V#consommable]{Consommable}, lorsque vous réalisez l’action, un message sera envoyé dans le tchat pour informer que vous avez consommé quelque chose.

Attention : si vous configurez un Équipement avec le Sous-type "consommable" ET avec une Résolution "consommable", à chaque utilisation de l'Équipement le système vous fera consommer deux unités d'un coup mais avec une seule fois l'Effet.

Application d'un effet

Lorsque qu'un effet est appliqué sur une cible, un message dans le tchat pour indiquer l'acteur qui a réalisé l'action, le nom de l'effet qui est appliqué et la cible de l'effet.

Description dans le chat

A chaque fois que vous voyez l’icône , cela signifie que vous pouvez envoyer des informations dans le tchat uniquement pour le MJ.

Dans tous les cas, le Message affichera l’image, le nom et la description de l’Objet.

Si l’Objet possède une Action/Effet associé alors un ou plusieurs boutons apparaîtront en bas du Message. Ils permettent d’interagir avec l’Objet (activer ou désactiver l’Effet, Attaquer ou lancer les Dommages, utiliser l’Objet…).

La description d’un objet dans le tchat est tout le temps envoyé uniquement au MJ et au PJ qui envoie le message. C’est pour cela que le fond du message est de couleur rouge, comme nous le verrons juste après.

Jet secret

Une distinction visuelle est faite dans le tchat en fonction de la visibilité du Message.

Message visible de tout le monde

Les messages visibles par tout le monde ont un fond "parchemin" en thème clair ou "noir" en thème sombre.

Message pour le MJ et moi

Lorsque le message est envoyé seulement au MJ et au joueur réalisant l’action, le fond du message devient rouge. Le contour du message est identique au messages publics.

Message juste pour moi ou juste pour le MJ

Lorsque le message est envoyé uniquement au MJ ou uniquement à "soi-même", le fond du message est rouge également ET le contour du message devient violacé en thème clair ou rouge en thème sombre.

Attention : le MJ est prévenu que vous avez envoyé un message juste pour vous même si le résultat est remplacé par des points d’interrogation. De la même manière si vous réalisez une action visible uniquement du MJ, vous verrez le message mais le résultat sera remplacé par des points d’interrogation.
